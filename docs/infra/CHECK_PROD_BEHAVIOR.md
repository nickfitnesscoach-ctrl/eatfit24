## Как протестировать правильно

### A. Через Telegram (Production)
1. Откройте Telegram Desktop
2. Найдите вашего бота (@YourBot)
3. Нажмите "Открыть приложение" / "Open App"
4. Тестируйте навигацию

### B. Локально с Debug (Development)
```bash
cd frontend
npm run dev
# Откройте http://localhost:5173/app
# Debug-режим активен, mock Telegram работает
```
